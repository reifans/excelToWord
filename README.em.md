# excelToWord
一个专注于标准格式文书处理的开源项目，旨在提高法律等从业人员处理大量文档的效率。 该项目通过搜索Excel表格和Word文档中的内容，建立对应关系，实现了自动化的文书生成过程。 通过简化从Excel到Word的数据转移，减轻专业人士在繁琐文书工作中的负担。
